# Student Management API

A simple ASP.NET Core Web API project for managing students using SQL Server, Entity Framework Core, and Repository Pattern.

## Features

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server Database
* Repository Pattern
* Database Migration
* CRUD Operations for Student
* Swagger API Documentation

---

## Technologies Used

* ASP.NET Core (.NET 10)
* Entity Framework Core
* SQL Server
* Swagger / OpenAPI
* Repository Pattern
